Detailed Description

All UI screens related to system configuration and user settings.

This group includes subpages for:

  • PI controller tuning (KP, KI, slew rate, max power)
  • Time/date configuration (RTC)

These screens allow the user to configure core system parameters through the rotary encoder interface.
